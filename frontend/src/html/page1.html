Pairwise Comparison

Below 20 one-on-one conversations are provided. For each conversation please select the best generated response. Between the generated responses only one response can be selected as best. You should judge a generated response as best according to 2 terms: Relevance and Fluency.

Please always keep in mind that:
  • Relevance evaluates whether the generated responses are on-topic with the dialogue context.
  • Fluency measures the grammatical correctness and readability of the generated responses.
